HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML stands for Hyper Text Markup Language
  • HTML elements tell the browser how to display the content
  • An HTML element is defined by a start tag, some content, and an end tag.
  • Some HTML elements have no content eg: line break (br). They are called empty elements and don't have an end tag.
  • Some HTML elements used to add content to a webpage include text, header, image, and list elements.
  • Other HTML element types used to divide and describe a page include meta-related elements, content division elements, and semantic elements.
  • An HTML attribute defines additional information about an element (ex:lang="en"), they are always defined in the beginning element, never the closing one.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git add [file]: add a file as it looks now to your next commit
  • git commit -m "[descriptive message]": commit your staged contentas a new commit snapshot
  • git branch: list your branches. a * will appear next to the currently active branch
  • git branch [branch name]: create a new branch at the current commit
  • git checkout: switch to another branch and check it out into your working directory
  • git merge [branch]: merge the specified branch's history into the current one
  • git log: show all commits in the current branch's history
